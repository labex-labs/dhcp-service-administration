Aprenda Administração do Serviço DHCP. Este módulo abrange a Instalação do Servidor DHCP ISC, Configuração de um Escopo de Sub-rede, Reservas de IP Fixo, Análise de Concessões (Leases), Solução de Problemas de DHCP. Você dominará estas habilidades essenciais do Linux através de laboratórios práticos e desafios do mundo real.
