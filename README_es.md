Aprenda la Administración del Servicio DHCP. Este módulo cubre la Instalación del Servidor DHCP ISC, la Configuración de un Alcance de Subred, las Reservas de IP Fijas, el Análisis de Asignaciones (Leases), y la Solución de Problemas de DHCP. Dominará estas habilidades esenciales de Linux a través de laboratorios prácticos y desafíos del mundo real.
