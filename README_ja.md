DHCP サービス管理について学習します。このモジュールでは、ISC DHCP サーバーのインストール、サブネットスコープの設定、固定 IP アドレスの予約、リース情報の分析、DHCP のトラブルシューティングを扱います。実践的なラボと実世界の課題を通じて、これらの必須の Linux スキルを習得します。
